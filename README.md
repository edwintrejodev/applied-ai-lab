# Applied AI Lab

Welcome to the Applied AI Lab repository. This repository serves as a centralized hub for various applied artificial intelligence projects, ranging from large language model training to computer vision and biometric systems.

## Project Portfolio

1. **[LLM MoE Finetuning](./llm-moe-finetuning)**
   Parameter-efficient fine-tuning strategies tailored for Mixture-of-Experts (MoE) architectures.

2. **[Distributed LLM Training](./distributed-llm-training)**
   Scalable training pipelines utilizing PyTorch Distributed Data Parallel (DDP).

3. **[Face Anti-Spoofing](./face-anti-spoofing)**
   Robust liveness detection models to secure biometric authentication systems against spoofing attacks.

4. **[Palm Recognition Model](./palm-recognition-model)**
   Deep learning approaches for palm-vein feature extraction and matching.

5. **[AI Model Optimization](./ai-model-optimization)**
   Techniques for making deep learning models smaller and faster, including quantization and pruning.
