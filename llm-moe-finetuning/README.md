# MoE Tuning Module

This sub-project focuses on adapting Mixture of Experts (MoE) models to specific domains using low-rank adaptations.

## Getting Started
Ensure you have the right environment setup. We highly recommend using `conda` or `venv`.

```bash
pip install -r requirements.txt
python src/moe_tuner.py --model_name "mistralai/Mixtral-8x7B-v0.1"
```

## Directory Details
- `config/`: Training configurations.
- `docs/`: Additional research notes.
- `src/`: Main execution code.
